### Primeiro Componente VUE - Catálogo de Livros

O código implementa um pequeno sistema de cadastro de livros utilizando Vue.js 3 com a API de composição (Composition API). Há um formulário simples que permite inserir título, autor e preço de um livro. Esses dados são vinculados ao estado reativo newBook via v-model, garantindo que o formulário esteja sempre sincronizado com os valores do objeto no script. Ao enviar o formulário, ela valida se todos os campos foram preenchidos e, em seguida, adiciona uma cópia do livro à lista books, limpando os campos do formulário após o cadastro. O código exibe dinamicamente a lista de livros cadastrados, caso existam. Cada item da lista mostra os dados registrados e inclui um botão de exclusão que é responsável por remover o livro correspondente da lista reativa books. Caso não haja livros cadastrados, é exibida uma mensagem indicativa.
